## 关于我们

尤物百科（U5book）是一个面向中文用户的全球创作者资料库与内容导航平台，专注于整理和收录来自全球热门社交媒体及内容订阅平台的创作者资料、平台信息、历史动态及相关内容索引。

平台通过人物库、分类库、标签库和专题库等方式构建完整的数据体系，帮助用户快速发现和了解全球热门创作者及平台生态。
